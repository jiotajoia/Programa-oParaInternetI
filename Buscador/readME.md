## Comandos

*readline-sync*

`npm i readline-sync`

`npm i types readline-sync`

*cheerio*

`npm i cheerio`

`npm install --save @types/cheerio`

*axios*

`npm i axios`

*diacritcs*

`npm i diacritics`

`npm install --save @types/diacritics`

## Instruções

Compilar normalmente o arquivo *typescript*.
Após compilado, executar o arquivo *javascript* gerado utillizando o *Node*.
